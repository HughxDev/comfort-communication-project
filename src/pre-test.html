Learning Outcomes

COMFORT™℠ is an acronym that stands for the seven basic principles of palliative care communication. This is module F (Family Caregivers) of the curriculum and you will learn about communication with family caregivers. After viewing this module, you will be able to:

  1. Define each caregiver type.
  2. Describe family communication patterns for caregivers.
  3. Respond appropriately to different health events tailored to different caregiver types.
